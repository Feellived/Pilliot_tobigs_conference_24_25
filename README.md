# 다중 알약 이미지와 공공 의약품 정보를 활용한 복약 참고 보고서 생성 시스템

투빅스 컨퍼런스 프로젝트 (유주형 · 한수영 · 김지우 · 조윤수)

---

## 프로젝트 의의

복약 안전은 환자 개인의 문제를 넘어 공중 보건의 핵심 과제다. OECD 보고에 따르면 전체 의료 과실의 18.3%가 투약 오류에서 비롯되며, 이는 개인의 건강뿐 아니라 막대한 사회적 비용으로 이어진다.

그럼에도 불구하고 실제 복약 현장은 여전히 취약하다. 처방전이나 약 봉투를 잃어버리거나, 여러 약을 한꺼번에 복용하면서 어느 것이 어떤 역할인지 기억하지 못하는 상황이 일상적으로 발생한다. 특히 고령층이나 만성질환자처럼 다약제를 장기 복용하는 경우, 이 문제는 더욱 심각하다.

---

## 배경 및 현황

기존의 알약 인식 연구는 대부분 **단일 알약의 분류 정확도**에 초점을 맞춰왔다. 색상·형태·인쇄 문자를 활용한 이미지 분류 모델이 주를 이루며, 실제 사용 환경보다는 통제된 데이터셋 기준 성능 향상에 집중된 경향이 있다.

반면 실제 사용자가 마주하는 상황은 다르다.

- 여러 종류의 알약이 한 화면에 함께 등장한다.
- 알약의 형태가 유사해 육안으로는 구분이 어렵다.
- 약 이름을 알아도 효능·주의사항·보관법 등 실질적인 복약 정보로 이어지지 않으면 의미가 없다.

이 간극을 해소하기 위해, 본 프로젝트는 알약 **검출 → 식별 → 공공 의약품 정보 연계 → 보고서 생성**으로 이어지는 통합 파이프라인 구현을 목표로 한다.

---

## 기대 결과

본 프로젝트가 완성되었을 때, 사용자는 다음과 같은 경험을 할 수 있다.

- 여러 알약이 담긴 사진 한 장으로 각 알약의 후보 약품명을 확인한다.
- 식별된 약품에 대해 효능, 주의사항, 복용법, 보관법 등의 정보를 요약된 형태로 제공받는다.
- 처방전이나 약 봉투 없이도, 손에 쥔 약에 대한 기본적인 복약 참고 정보를 스스로 확인할 수 있다.

궁극적으로 이 시스템은 단순한 이미지 분류 도구가 아니라, **알약 인식 결과를 사용자 맥락에서 의미 있는 정보로 재구성하는 서비스형 AI**로서의 가능성을 보여주는 것을 목표로 한다.
